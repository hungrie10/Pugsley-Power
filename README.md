# 🚀 Pusgley

Pusgley is a productivity software designed to help you stay consistent, organized, and fulfilled.  
It combines essential daily tools into one focused workspace so you can track progress, build streaks, and see your growth clearly.

---

## ✨ Features

### 📅 Calendar
Plan your days, visualize your schedule, and stay ahead of deadlines.

### 🌤 Weather Component
Get real-time weather updates to plan your day effectively.

### ✅ To-Do List
Create, manage, and complete daily tasks with ease.

### 🔥 Weekly Streak Tracker
Build consistency by maintaining a streak throughout the week.

### 📊 Daily Completion Tracker
See how much of your daily tasks you’ve completed with a clear progress indicator.

---

## 🎯 Goal

Pusgley is built to make productivity feel rewarding rather than overwhelming.  
Instead of just listing tasks, it helps you:

- Stay consistent  
- Track daily progress  
- Build habits  
- Feel accomplished  

---

## 🛠 Tech Stack

- Frontend: HTML, CSS, JavaScript  
- Backend: Node.js  
- Database: MongoDB  
- APIs: Weather API
